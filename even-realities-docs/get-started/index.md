# Get Started

*From zero to a Hello-on-glasses app in 15 minutes - plus the deep-dive pages for each step.*

Source: https://hub.evenrealities.com/docs/get-started/

> **Last updated:** 2026-06-11

Start here if you're new to Even Hub. The [Quickstart](https://hub.evenrealities.com/docs/get-started/quickstart/) page is the condensed one-page path; every linked deep dive expands the step it points to.

- **Without glasses yet?** Complete [Sign in](https://hub.evenrealities.com/docs/get-started/quickstart/sign-in), [Install Node.js & npm](https://hub.evenrealities.com/docs/get-started/quickstart/install-node), [Install Even Hub tooling](https://hub.evenrealities.com/docs/get-started/quickstart/install-tools), and [Your First App](https://hub.evenrealities.com/docs/get-started/quickstart/first-app) - all doable without hardware.

- **Have your G2?** Continue through [Hardware](https://hub.evenrealities.com/docs/get-started/quickstart/hardware) for pairing, firmware, Developer Mode, and firewall setup.

- **Want the big picture?** [Architecture](https://hub.evenrealities.com/docs/get-started/architecture) explains how your code reaches the glasses.

- **Want a head start?** [Templates](https://hub.evenrealities.com/docs/get-started/quickstart/templates) skips the manual scaffold with a complete starter project + first-party component library.
